# Cancer Hallmarks Network

This repository contains the multiscale cancer hallmarks network developed for a PhD thesis in complex systems modelling.

## Contents

- GraphML network model
- Boolean network (.bnet) file
- Hallmark annotations
- Simulation scripts
- Supporting metadata

## Software

Networks were analysed using:
- Cytoscape
- R
- MaBoSS

## File Formats

### GraphML
Used for network topology and visualisation.

### BNET
Used for Boolean simulations and attractor analysis.

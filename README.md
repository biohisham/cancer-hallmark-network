# Cancer Hallmark Network

This repository contains the multiscale cancer hallmark network developed for a PhD thesis in systems cancer biology.

## Contents

- GraphML network models
- Boolean network (.bnet) files
- Hallmark annotations
- Simulation scripts
- Supporting metadata

## Software

Networks were analysed using:
- Cytoscape
- R
- MaBoSS

## File Formats

### GraphML
Used for network topology and visualisation.

### BNET
Used for Boolean simulations and attractor analysis.
